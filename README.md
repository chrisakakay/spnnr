# spnnr
Simple terminal spinner class

# Install

```bash
npm install spnnr --save
```

# Usage

```javascript
const Spnnr     = require('spnnr');
const spinner   = new Spnnr(`Searching for files`);
// Do something
spinner.stop(`Search done.`);
```
